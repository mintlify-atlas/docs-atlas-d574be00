## Installing skills

Skills are distributed through plugin packages. To install a skill:

```bash
/plugin install <plugin-name>@<marketplace-owner>
```

For example, to install the GitHub PR message generator:

```bash
/plugin install github-write-pr-message@fuongz-skills
```

Once installed, you can invoke skills using their slash commands:

```bash
/<skill-name> [arguments]
```
