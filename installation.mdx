## Add the marketplace

<Steps>
  <Step title="Open Claude Code">
    Launch Claude Code in your terminal or IDE.
  </Step>
  
  <Step title="Add the marketplace">
    Run the following command to add the Skills Marketplace:
    
    ```shell
    /plugin marketplace add fuongz/skills
    ```
    
    This registers the marketplace with Claude Code, making all available skills discoverable.
  </Step>
  
  <Step title="Verify installation">
    The marketplace is now added and you can browse available skills. Use the plugin commands to explore what's available.
  </Step>
</Steps>

<Note>
  After adding the marketplace, you'll need to install individual skills before using them. See the [Quick start guide](/quickstart) for a complete example.
</Note>

## What happens when you add the marketplace

When you add the Skills Marketplace:

- Claude Code registers the marketplace repository (`fuongz/skills`)
- You gain access to all skills published in the marketplace
- Skills can be installed individually using their unique identifiers
- Updates to the marketplace become available automatically
